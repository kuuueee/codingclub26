Loops intro

 Hopefully from your scratch days, you remember loops.

            There are a few different types of loops you can use, but I find my favourite is the for loop.

            Here is what a for loop could look like in scratch.

            We create a variable i, set to 0. Then we repeat all the steps inside the loop until i = 5.

            Within the loop, every iteration we check if i > 2. If it is turn anticlockwise 15 degrees.

            Then we always move one step and increase the value of the variable i by 1.
        
 
            This loops is literally translated to the following js.

            var direction = 0;
            for (var i = 0; i < 5; i++)
            {
                if (i < 2) {
                    direction = (direction % 360) - 15;
                }
                console.log("When i = " + i + " - Moved one step " + direction + "°");
            }

            To break that down, we set a variable for direction that we need to keep track of and initialize it to zero.

            We start the for loop by saying; 
                - Set variable i equal to zero.
                - Run the everything inside the bracket while i is less than 5.

            By saying this; the loop will run 5 times: when i = 0, 1, 2, 3, 4. Notice i does not equal 5, and our for loop explicitly states while i is less than 5, so never while i = 5.
