Randomizer

I think that we should just play with some javascript for a bit.

 The goal here is to create a webpage that does a random tier list.
              
              You're going to define the order of tiers, and their name.
              I'm going to use the example of S-A-B-C-D tiers
                You will define different names for your tiers, don't worry.
        

        
            To get started, please note the codingclub26/mrc/project_template/random folder.

            In Chrome, open Randomizer Tutorial
            In Visual Studio Code open random.js

            You will be making all of your edits to random.js
        


        
            Notice that the currently functionality makes it so that one tier will only have one item.

            Personally I think Hockey and Soccer are S tier sports so I would like to fix this.

            I want to be able to have both items appear in one tier.

            Your goal today is to help me get that functionality.
        



        
            After you go throught the document, you will be faced with the opportunity to edit the function newRandomTiers in your random.js file.

            This is the whole tutorial, but beware, it's a bit of a doozy please start asking around for help.
