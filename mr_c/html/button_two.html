We ended last session with a button that counts up.

To be able to craft new ideas, let me walk you through how the components work

	Take a look at the following HTML code that produces a the counter button in your program:

		<button type="button" onclick="hello()">Click me</button>  
			
	Now let's look at a new button, this time it's empty : 

		<button></button>  <--- There is a button here

	Here are the elements of the button and what they do:
			
		Element text     : The text that appears on the button. Our example says "Click me" 
		Element type     : The type of button that this is, such as reset, submit element types
		Element event    : How the function will react. There are many events, in our example it will be when the button is clicked.
		Element function : The function that is executed when the button is pressed. Our example says "hello()"
