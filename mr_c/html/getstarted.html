Your website and coding environment

Finder

	Use your finder app to navigate your webiste's directory.

Visual Studio Code

	This is your text editor that you will use to write your html, css, and js code in.
	Open Visual studio code, then navigate to open project. Select your project.

Browser

	Chrome is a good browser to use. You will use your browser to view your website!.
	A handy tip is to create a bookmark for your index page, that way it is easy to get to.
	To open your project in your browser, find it in your finder.
	The browser will also be the place you visit when you want to search the internet for media
	

				You will use a combination of these tools to slowly add features to your website and customize it as you please.
