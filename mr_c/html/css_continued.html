What CSS Does

We now hopefully know at least these three things 

            1. HTML handles text based elements
            2. Javascript is how you can programatically change these elements
            3. CSS is the layer applied to these elements that handle style

I will demostrate below with a few things.


The following element is a picture element. I've manually applied some css values, but you can manipulate the values with the text boxes below. 

Note all of these need to be valid before it will change



In this next tutorial you start to interact with how html elements interact with the css you apply to it

            High level instructions:
                - Find an image online from one of the sources posted in the lab (I used wikimedia commons to find this sick image of Danny Devito)
                    - Give your image a name
                - In a new folder save the image to the name of your folder
                    - You will have a new folder named image_name
                        in it will be these files:
                            {image_name}.html
                            {image_name}.js
                            {image_name}.css
                            {image_name}.jpg/png/bitmap/ any image format.
                - Create buttons that alter the way that this image is displayed on your site
                    Ideas for this
                    - Move the image around a grid
                    - Flip the image upside down
                    - Rotate the image
        


            Lower level instructions:
                - Go to Wikimedia commons and browse for an image
                - Save the image to a new folder
                - Name the image and the folder the same name and put your image in your folder
                - You will have a folder - inside the folder is an image and both will have the same name
                - Using danny as an example:
                    danny
                        danny.jpg
                - Create the three files that will host your web things
                    danny.html
                    danny.js
                    danny.css
                
                - Copy any html page that you've already created and paste the entire document in {image_name}.html aka danny.html
                    - Remove everything inside the body of your document
                    - Edit the Title to reflect the image_name aka Danny
                    - In head
                        - Edit or add the script tags for danny.js
                            <script src="danny.js"></script>
                        - Edit or add the link tags for danny.css
                            <link rel="stylesheet" href="danny.css">

                

                - In the body of danny.html
                        - Add the image element:
                                <img id="danny" src="..\media\img\Danny-devito-yearbook-photo.jpg" alt="Danny Devito Yearbook Photo">
                    
                    - Notice the id for javascript
                    - Notice the src for the location of where you downloaded your image.
                        - If you don't see an image look back to your folder structure
                    
                    - You will have to change the id from danny to your image Name

                    - Add a button to your danny.html that correlates to a function in your danny.js
                        - name it doSomething
                        <button type="button" onclick="doSomething('danny')">If you click me i'll do something!</button> 
                        - * Change the id from danny to your image name
                        
                

                - In your danny.css file

                        - Create image rules for your new image
                        - The hash/pound/sharp/number sign # in your css file correlates to the id.

                    -Danny looks like this, but you can alter to fit your image. Make sure to change the #name

                    - #danny {
                        width: 500px;
                        height: 600px;
                        border: red 2px solid;
                    }
                

                - In your danny.js file
                    
                    - Define the function doSomething:

                - Copy this into your js file:

                    function doSomething(imgId) {
                        var imageElement = document.getElementById(imgId);
                        imageElement.style.width = "100%";
                        imageElement.style.height = "100%";
                        imageElement.style.objectFit = "cover";
                    }

                

                - You just made a button that does something to your image.
                - See what else you can make to edit this image!
                - Follow the same steps of making a button in html and javascript but change the name of the function to "doSomethingElse"
                - Make the button do something! Feel free to ask!
